# UI/UX Polish

Iterative UI/UX polishing workflow for web applications. Achieve Stripe-level visual polish through multiple passes — each iteration adds incremental improvements that compound dramatically.

## Install

```
/plugin marketplace add JuanMarchetto/agent-skills
/plugin install ui-ux-polish@agent-skills
```

Or via [skills.sh](https://skills.sh):
```bash
npx skills add JuanMarchetto/ui-ux-polish-skill
```

Or manually:
```bash
git clone https://github.com/JuanMarchetto/ui-ux-polish-skill.git
cp -r ui-ux-polish-skill ~/.claude/skills/ui-ux-polish
```

## What's Inside

- The exact polish prompt that produces Stripe-level results
- Why the prompt works (agreement, desktop/mobile separation, high anchors)
- Iteration protocol for single and multi-agent workflows
- What the model typically improves (visual polish, interaction, mobile, desktop)
- When to use vs when NOT to use
- Alternative scrutiny prompt for deeper passes
